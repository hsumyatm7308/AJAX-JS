=>AJAX  (Asynchronous Javascript And XML)
  
  - not a programming language 
  - a technique for accessing web server from web page 
  - allows the client to send data and receive data in an asynchronous manner 
  - can be used with any type of script, including Javascript



AJAX use a combination of:
   - a broswer built-in XMLREquest (to request data from a web server)
   - Javascript and HTML DOM (to display or use the data)



=>How to Work

   - Create XMLHTTPRequest Object by Javascript
   - send request that Object to web server 
   - Server processes the request and reponse to web page 
   - Display updated page by Javascript
