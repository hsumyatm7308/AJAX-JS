=>XMLHTTPRequest Object  (Key of AJAX)

  - Create an XMLHTTPRequest Object
  - Define a callback function 
  - Open the XMLHTTPRequest Object
  - Send a request to the server 

  ## How To Use 

   1. Create an XMLHTTPRequest Object
      syntax =>   vairable = new XMLHTTPRequest();

   2. Define a callback function 
      syntax =>   xhtml.onload = function(){ ##code to be executed }

   3. Open the XMLHTTPRequest
      syntax =>  xhtml.open('method','open file')
    
   4. Send request 
      syntax => xhtml.send();
